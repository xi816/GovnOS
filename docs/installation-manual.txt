GovnOS installation guide manual

1. Setting up the drive
Set up a drive for installation
You can change the drive size, memory size, **VAR size and **LEGV size
In this example provided:
	Drive size -- 4096 (4KB)
	Memory size -- 4096 (4KB)
	**VAR size -- 8128 (7.9375KB)
	**LEGV size -- 64 (0.0625KB)
# ./memory-gen DRIVENAME 4096 4096 8128 64
	Are you sure? (Y/N): Y
	Drive created.

2.1. User
Add a user to use in GovnOS
$ adduser
  Enter user: USER
  Add a password: 1234
  Add? (Y/N, default: Y): Y

2.2. Root
Change the root password
$ passwd
  Enter new root password: 69420

2.3. Hostname
Change the hostname to what you want
$ hostname
  Enter hostname: POCO

2.4. Configuration
Now, generate the config files
$ config-gen

3. Finish
That's it! GovnOS is now installed and you can shut it
down.
$ exit
Then run it again using ./run DRIVENAME



$ -- GovnOS terminal command
